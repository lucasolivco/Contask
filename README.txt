# 🧩 Task Manager

Um gerenciador de tarefas completo com autenticação e controle de usuários. Este projeto fullstack permite que um gerente crie, atribua e acompanhe tarefas de forma organizada e segura.

---

## 🚀 Tecnologias Principais

- **TypeScript**
- **Node.js** + **Express**
- **React** (frontend em desenvolvimento)
- **MongoDB** ou outro banco de dados (configurável)
- **dotenv**, **helmet**, **cors**, **rate-limit**, entre outros

---

## ⚙️ Funcionalidades

- Cadastro e login de usuários com autenticação
- Criação, listagem e atualização de tarefas
- Atribuição de tarefas a usuários
- Proteção contra abusos com rate limiting
- Middleware global de segurança e tratamento de erros
- API estruturada e modular

---
